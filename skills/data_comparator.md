# Data Comparator Skill

When asked to compare data across two or more files or sources:

1. **Extract key metrics** from each source independently before comparing.
2. **Build a comparison table** in Markdown format showing: Metric | Source A value | Source B value | Match?
3. **Identify root causes** of mismatches — could be rounding, different date ranges, different definitions of the same term, or genuine errors.
4. **Confidence rating**: Rate your confidence in the comparison as HIGH / MEDIUM / LOW and explain why (e.g., "LOW because the PDF does not specify whether revenue is gross or net").
5. **Verdict**: State clearly whether the documents are consistent, inconsistent, or inconclusive.
